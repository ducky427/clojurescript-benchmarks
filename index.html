ClojureScript micro benchmarks over different releases over time

These benchmarks have been run through simple benchmarking code and not through Benchmark.js. Hence expect noise in these numbers and possibly a large margin of error.

Setup

These benchmarks were run on Mac OS X 10.11.5 with the following engines:

  • Javascript Core (JSC): Bundled with Safari Tech Preview version 9.1.2 (11601.7.4, 11602.1.35)
  • V8 (V8): v5.1.281.47
  • SpiderMonkey (SM): JavaScript-C50.0a1

The code for running these benchmarks is here.
